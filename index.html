Core Architecture

○

Decision-Gated Protocol

No body change = no next step. The body is the authority, not the mind, not the schedule, not the therapist, and not the protocol itself.

↑

Structured Escalation

When first-line regulation fails, escalation is not failure — it is the next step. DGAEP provides a 6-level autonomic escalation pathway.

◆

Pattern Recognition

HRV tracking, load domain detection, and weekly pattern analysis. The pattern matters, not the individual day.

◎

Working Memory Support

Autonomic dysregulation compromises working memory. The tool remembers for you — tracking steps, states, and changes when cognition is overloaded.

⊗

Red Line Thresholds

Non-negotiable structural load limits. Crossing a red line is not failure — it is the system signaling that protective action is required.

∞

Continuous Validation

Built on 4.5 years of continuous HRV data, tested across cognitive, sensory, emotional, social, and executive load domains.

Yve Bergeron

The nervous system is the last interface. Everything else — sensory processing, executive function, emotional regulation, social reciprocity — runs on top of it. If the autonomic layer is dysregulated, nothing built on top of it will function as intended.
